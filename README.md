# 🎓 西安小升初智能评估系统 2025

## 🌟 项目简介

**西安小升初智能评估系统**是一款基于人工智能的个性化小升初学校推荐平台，为西安地区的学生和家长提供专业、精准的升学规划服务。

### ✨ 核心特色
- 🤖 **AI智能推荐** - 基于大模型的个性化学校推荐
- 📊 **多维评估** - 6大能力维度全面分析
- 🎯 **精准匹配** - 基于户籍、居住、预算等真实因素
- 📍 **本地化数据** - 完整的西安区县街道数据库
- 💰 **经济适配** - 考虑家庭预算的学校推荐

## 🚀 快速开始

### 环境要求
- Node.js 14+
- 现代浏览器（Chrome、Firefox、Safari）

### 安装部署

1. **克隆项目**
```bash
git clone <项目地址>
cd 西安小升初
```

2. **安装依赖**
```bash
npm install
```

3. **启动服务**
```bash
npm run dev
# 或直接打开 index.html
```

4. **访问应用**
```
https://xian-school-app.vercel.app/
```

## 🎯 功能特色

### 1. 智能评估流程
| 步骤 | 功能 | 说明 |
|------|------|------|
| 1️⃣ | 能力评估 | 6大维度能力测评 |
| 2️⃣ | 家庭信息 | 户籍、居住地、房产情况 |
| 3️⃣ | 教育偏好 | 学校类型、预算、教育理念 |
| 4️⃣ | AI分析 | 大模型深度分析 |
| 5️⃣ | 个性推荐 | 5所学校精准推荐 |

### 2. AI推荐策略
- **🏃‍♂️ 冲刺校** (2所) - 高匹配度，竞争激烈
- **⚖️ 稳妥校** (2所) - 适中匹配，录取概率高  
- **🛡️ 保底校** (1所) - 确保入学，安全选择

### 3. 支持的大模型
- ✅ 阿里百炼 (qwen-plus)
- ✅ OpenAI (gpt-4o-mini)
- ✅ DeepSeek (deepseek-chat)
- ✅ Google Gemini (gemini-pro)

## 🎮 使用指南

### 第一步：能力评估
- 对6个能力维度进行评分（1-5分）
- 包括：学业成绩、综合素养、学习习惯等

### 第二步：填写家庭信息
- 选择户籍所在区和街道
- 选择实际居住区和街道
- 填写房产情况

### 第三步：教育偏好
- 民办学校意向
- 家庭预算范围
- 学业规划目标
- 学生特长选择
- 教育理念偏好

### 第四步：生成报告
- 查看能力雷达图
- 获取AI学校推荐
- 阅读详细分析报告
