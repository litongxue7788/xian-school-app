✨ 项目特点
本工具深度融合 AI 技术，为您提供一站式的升学规划体验，成为您的私人升学军师。

[🎯] 高度个性化: 告别通用模板！通过包含学生能力、家庭状况、教育理念、风险偏好在内的 六维画像问卷，为每个家庭建立独一无二的档案。

[🧠] 全方位 AI 赋能: AI 不仅是聊天工具，更是贯穿全程的决策引擎：

AI 择校推荐: 结合您的家庭画像与学校官方数据，智能生成“冲刺”、“稳妥”、“保底”三大梯度择校方案。
AI 政策解读: 您的户籍、房产具体对应第几顺位？有何风险？AI 将复杂政策翻译成清晰易懂的专属解读。
AI 升学助手: 内置“小喵”聊天机器人，7x24小时随时解答您关于升学政策、学校情况的各种疑问。
[📊] 一键生成综合报告: 完成评估后，系统将自动生成一份精美的综合报告，包含：

能力雷达图: 强弱项一目了然。
择校策略表: AI 推荐的学校、理由及成功率预估。
升学时间轴: 关键节点与待办事项，助您从容规划。
核心策略建议: 基于您家庭情况的专属建议总结。
[⚙️] 灵活的双模式设计:

AI 增强模式: 连接您自己的 API Key，解锁全部智能分析功能，体验最佳。
本地模式: 无需任何配置，也可使用基础的问答和保底推荐功能。
[📤] 报告导出与分享: 支持将评估报告导出为 PDF 格式，方便您随时查阅、打印或与家人分享。

🚀 如何使用本系统 (三步走)
第一步：激活 AI (配置)

打开网站后，点击页面右上角的 “本地模式” 按钮。
在弹出的配置面板中，填入您的 API Key 和 APP ID。
选择对应的 AI 服务商（例如：阿里百炼/通义）。
点击 “保存并测试”。当提示“连接成功”时，AI 大脑即被激活！
第二步：完成评估 (输入)

跟随引导，依次完成 “学生能力评估” 和 “家庭需求评估” 两大模块的问卷。
您的回答越贴近真实情况，AI 给出的建议就越精准。
第三步：生成报告 (输出)

完成所有问卷后，点击最后的 “生成评估报告” 按钮。
系统将立即为您呈现一份包含所有分析结果的、为您量身定制的专属报告。
在任何时候，您都可以点击右下角的 猫咪图标 🐱 ，随时向 AI 助手提问。

部署与开发
您可以非常轻松地将此项目一键部署到 Vercel 平台。

Fork 本仓库: 点击项目页面右上角的 "Fork" 按钮。
在 Vercel 中导入并部署:
登录 Vercel，选择 "Add New..." -> "Project"，然后选择您刚刚 Fork 的仓库。
Framework Preset: 确保选择 Other 。
Build and Output Settings: 全部留空。
Environment Variables: 务必配置！ 这是激活 AI 功能的关键。
BAILIAN_API_KEY : 你的阿里百炼 API Key
BAILIAN_APP_ID : 你的百炼应用 APP ID
(或 OPENAI_API_KEY , DEEPSEEK_API_KEY 等)
点击 "Deploy"，等待 1-2 分钟即可完成。
